# Arous – Wedding Vendor Finder

## Overview

Arous is a web application that helps brides find wedding vendors in one place. Users can browse vendors, view their profiles, check availability, save favorites, and contact vendors directly through WhatsApp.

---

## Problem

Planning a wedding often requires searching through Instagram, Facebook, WhatsApp groups, and personal recommendations to find reliable vendors. This process is time-consuming and makes it difficult to compare different options.

Arous brings all wedding vendors together in one organized platform, making the search process simple and efficient.

---

## Target Audience

The application is designed for:

- Brides planning their wedding.
- Wedding vendors who want to present their business online.
- Users looking for a simple and organized way to compare wedding services.

---

## Competitors

Current alternatives include:

- Instagram
- Facebook
- WhatsApp Groups
- Google Search
- Manual recommendations from friends and family

---

## What Makes Arous Different?

- Wedding vendors are organized in one platform.
- Easy filtering and browsing.
- Vendor availability is displayed.
- Favorite vendors can be saved.
- Direct communication through WhatsApp.
- Simple, modern, and user-friendly interface.

## Installation

Clone the repository:

```bash
git clone https://github.com/areen16/Arous-App.git
```

Go to the project folder:

```bash
cd Arous-App
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
